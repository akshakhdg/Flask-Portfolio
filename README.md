# Flask Portfolio Website

A personal portfolio website built with Python and Flask to showcase my projects, skills, and contact information.

## Features

- About, Skills, Projects, and Contact sections
- Project cards with GitHub/live links
- Flask routing and template rendering
- Organized static assets and templates

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Git & GitHub

## How to Run Locally

```bash
git clone https://github.com/YOURUSERNAME/flask-portfolio.git
cd flask-portfolio
pip install -r requirements.txt
python3 app.py
